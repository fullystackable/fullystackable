# fullystackable brand assets

This directory is reserved for static brand assets used by the fullystackable app itself.

Do not store client or managed brand assets directly in `public/brand`.

Managed brand assets should be tracked as database asset records and will usually be linked from systems such as Google Drive, Canva, Dropbox, external URLs, or Supabase Storage later.

Only static fullystackable app assets belong here, including app logos, icons, social previews, and fallback brand imagery.
